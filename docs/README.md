# Docs 文件夹说明

本文件夹包含项目相关的所有文档。

## 文档列表

### 准备阶段文档

1. **Problem_D_准备工作清单.md**
   - 内容：完成Problem D需要做的准备工作
   - 包括：数据准备、建模方法、技术工具、时间规划等

2. **数据分析和需求清单.md**
   - 内容：详细的数据需求分析
   - 包括：已有数据、缺失数据、数据来源、获取方法等

### 数据获取文档

3. **数据获取详细指南.md**
   - 内容：详细的数据获取步骤和方法
   - 包括：NYC Open Data使用方法、API使用、手动下载指南等

4. **数据获取快速开始.md**
   - 内容：快速开始指南
   - 包括：最快方案、数据清单、工具准备等

5. **数据获取状态.md**
   - 内容：数据获取状态摘要
   - 状态：已过时，数据已收集完成

### 流程说明文档

6. **数据阶段完整流程说明.md**
   - 内容：数据阶段的完整流程和代码说明
   - 包括：每个步骤的详细代码、执行结果、技术要点等

## 文档使用建议

- **新手入门**：先看 `Problem_D_准备工作清单.md`
- **数据收集**：参考 `数据获取详细指南.md`
- **了解流程**：查看 `数据阶段完整流程说明.md`
- **数据说明**：查看 `data/README.md`
