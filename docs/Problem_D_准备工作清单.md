# 2025 CQUICM Problem D: 垃圾收集优化问题 - 准备工作清单

## 一、问题理解与需求分析

### 1.1 核心问题
- **目标**：优化曼哈顿垃圾收集调度，解决多个目标（效率、公平性、健康风险、老鼠问题）
- **范围**：曼哈顿12个DSNY指定卫生区（MN1-MN12）
- **约束**：街道狭窄、车辆和行人多、工会化工人（每车2人）

### 1.2 五个核心任务
1. **资源分配策略**：优化垃圾收集频率和分布，确定需要更多收集的区域，估算卡车数量
2. **效率与公平性评估**：建立定量指标，分析效率与公平性的权衡
3. **中断场景分析**：应对车辆故障、恶劣天气、节假日、垃圾量激增等
4. **老鼠问题**：分析垃圾与老鼠数量的相关性，优化早晚收集时间分配
5. **垃圾桶替代塑料袋**：分析新政策对老鼠数量和卡车需求的影响

## 三、建模方法准备

### 3.1 优化模型
- **车辆路径问题（VRP）变体**
  - 多目标优化（效率 vs 公平性）
  - 时间窗约束
  - 容量约束
  - 动态调度（应对中断）

### 3.2 分析方法
- **聚类分析**：识别需要高频收集的区域特征
- **回归分析**：垃圾量与老鼠数量的关系
- **敏感性分析**：政策变化的影响
- **鲁棒性分析**：中断场景下的表现

### 3.3 评估指标设计
- **效率指标**：
  - 卡车利用率
  - 总行驶距离/时间
  - 收集频率与需求的匹配度
  - 成本效益比

- **公平性指标**：
  - 各区域收集频率的方差/基尼系数
  - 低收入区域的收集质量
  - 垃圾暴露时间的公平性

- **健康/环境指标**：
  - 垃圾暴露时间
  - 老鼠投诉数量变化
  - 公共卫生风险评分
